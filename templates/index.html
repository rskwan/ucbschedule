UCB Schedule of Classes by Ronald Kwan

API

We provide an API for data obtained from scraping the UC Berkeley Schedule of Classes. Currently, data from Fall 2014 (starting from 2014-04-06) is in the database. Examples:

  • /api/departments?page=1: returns first page of departments
  • /api/departments/36: returns department with id=36 (COMPSCI)
  • /api/departments/36/courses?page=1: returns first page of courses with department COMPSCI
  • /api/courses/610: returns course with id=615 (COMPSCI 176, Fall 2014)
  • /api/sections/2001: returns section with id=2001 (COMPSCI 176 LEC 001, Fall 2014)
  • /api/sections/2001/instances: returns instances (where an instance is a section's data for a particular day) of COMPSCI 176 LEC 001, Fall 2014
